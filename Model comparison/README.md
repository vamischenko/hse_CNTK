# Model comparison

В данном разделе приводится сравнение качества обучения различных моделей из примера https://github.com/Microsoft/CNTK/wiki/Hands-On-Labs-Language-Understanding.
Каждая модель обучалась 20 эпох.

### sluSimple

Базовая модель примера.
Ошибка на тестовой выборке: 2.276%

### sluLookAhead

Модель, учитывающая следующее слово в LSTM слое.
Ошибка на тестовой выборке: 1.921%

### sluBiDirectional

Модель, использующая двунаправленный LSTM слой.
Ошибка на тестовой выборке: 1.830%

Более подробная информация о процессе и результатх обучения содержится в log-файлах.
